How to Set Up and Configure Menu Item Versions

Menu Item Versions are required if you are using Barcode Parsing or Menu Versions.

Creating/Configuring a Menu Item Version

To set up a Menu Item Version, select the RF-SMART tab, then select Client and Functions from the drop-down menus. The RF-SMART Menu Item List page opens.

  • Find the function you want to create a Menu Item Version for and select View.

The Function's RF-SMART Menu Item page will open.

  • Select Versions from the secondary navigation bar, then the New RF-SMART Menu Item Version tab.

The RF-SMART Menu Item Version page will open

  • Fill in Name field at the top.
  • Select the Import From Template tab.
  • Click Submit.

Note: We recommend importing Screen Definitions instead of adding them individually. To add a definition manually, fill in the respective fields. Select Add to create additional Screen Definition lines.
